# Demons SMP Launcher

A modern, web-based Eaglercraft launcher with a space-themed design.

## Features
- **Animated Star Background**: Smooth, falling star effect using HTML5 Canvas.
- **Minecraft Style**: Pixel fonts and glowing borders for an authentic feel.
- **Client Support**: Dedicated pages for multiple Eaglercraft versions.
- **Mobile & Chromebook Ready**: Responsive design that works on any device.
- **Full Screen Support**: Clients launch in a full-screen iframe.
- **GitHub Pages Compatible**: Ready to be deployed as a static site.

## Clients Included
1. Pixel Client 1.12.2
2. Tuff Client Beta 1.12.2
3. Wasm-GC Tuff Client Beta
4. Wasm-GC Pixel Client 1.8.8
5. Wasm-GC 1.20.4

## Customization
To change the client URLs, edit the `src` attribute of the `<iframe>` tag in the respective files within the `clients/` folder.

## File Upload
The launcher includes a file upload area on the home page for users to manage their Eaglercraft files locally.
